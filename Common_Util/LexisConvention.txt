一些词汇的约定: 
注: <约定> 创建于 2024.10.19, 在此之前的注释中可能会有不按此约定来的, 后续遇到了再调整过来.

比较相关: 
相同 => 指比较对象是同一个东西
相等 => 指比较对象完全相等 (通常情况下, 两个相同类型的变量, 值均为 null 的情况下, 应被视为相等)
等价 => 指比较对象不完全相等, 但是在特定的情况下可以视为相等
(两个东西, 相同时必然相等, 相等时必然等价)
同构等位相等(或等价) => 用于比较数组, 列表, 树结构等具有一定结构的一系列对象, 在结构等价的情况下(比如一方是数组, 一方是列表, 两者均为可视为有序定长一维结构, 这种情况可被视为结构等价), 不同结构对象中各个相同位置的项均相等(或相同)
严格同构等位相等(或等价) => 用于比较数组, 列表, 树结构等具有一定结构的一系列对象, 在结构相等的情况下(比如一方是数组, 一方也必须是相同类型的数组, 这种情况可被视为结构相等), 不同结构对象中各个相同位置的项均相等(或相同)
异构等位相等(或等价) => 用于比较数组, 列表, 树结构等具有一定结构的一系列对象, 在以特定方式遍历时(比如一方是数组, 按索引顺序遍历, 一方是树, 按先根次序遍历), 在子项总数相等的情况下, 相同遍历索引的项均相等(或相同)
